# User stories

Use this template when writing the user stories:

*Som en spelare så vill jag lägga min pjäs på tom ruta

*Som en spelare så vill jag veta att det är min tur

*Som en spelare vill jag kunna lämna spelet

*Som en spelare får jag veta om jag har vunnit

*Som spelare vill jag veta om spelet är fullt 

*Som en spelare så ska jag inte kunna lägga en pjäs på motspelarens tur

*Som en spelare ska man kunna se status på spelplanet

Create a new Markdown file for each userstory and create a toc.yml in the userstories folder, and add each userstory to the TOC (table of content).

Read more on userstories at [https://www.mountaingoatsoftware.com/agile/user-stories
